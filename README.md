# DaisyUI Todo List


## Features
- Javascript
- Tailwindcss
- DaisyUI
- Hero Patterns
- Dark/Light mode theme toggle (!)
- Google Fonts

- crypto.randomUUID() method for the task.id and note.id to help with errors of using the index or tasks.length for the id on
